# Propane Puppet Module for Boxen

[![Build Status](https://travis-ci.org/boxen/puppet-airfoil.png)](https://travis-ci.org/boxen/puppet-airfoil)

Installs Propane App

## Usage

```puppet
include propane
```

## Required Puppet Modules

* boxen
* stdlib

## Developing

Write code.

Run `script/cibuild`.
