# Propane Puppet Module for Boxen

[![Build Status](https://travis-ci.org/boxen/puppet-propane.png)](https://travis-ci.org/boxen/puppet-propane)

Installs Propane App.

## Usage

```puppet
include propane
```

## Required Puppet Modules

* boxen
* stdlib

## Developing

Write code.

Run `script/cibuild`.
